# conversational-analytics-api-demo

This demo is made for workshop 'Building a Data Chatbot: Intro to Conversational Analytics API (with ADK)'
for DevFest 2025

Unlock the power of your data by making it accessible through natural language. In this hands-on workshop, you will learn how to build an intelligent, AI-powered data agent that can answer questions about your structured data in BigQuery. Using the the Conversational Analytics API's Python SDK and Agent Development Kit (ADK), attendees will connect to a BigQuery data source to enable natural language querying.
